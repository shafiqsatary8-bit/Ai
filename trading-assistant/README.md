# ⚡ دستیار تصمیم‌گیری معاملاتی شفیق

## درباره این اپلیکیشن

یک تحلیل‌گر معاملاتی حرفه‌ای و هوشمند برای تریدرهای رمزارزی. این برنامه بر اساس استراتژی‌های تحلیل تکنیکال جدید، تصمیمات منطقی برای ورود به معامله فراهم می‌کند.

## 🎯 ویژگی‌های اصلی

✅ **تحلیل اسلاست و دقیق** - استفاده از شاخص‌های معتبر بازار  
✅ **داده‌های زنده** - اتصال مستقیم به Binance API  
✅ **رابط کاربری فارسی** - تمام توضیحات و برچسب‌ها در فارسی  
✅ **طراحی حرفه‌ای** - دارک مد با رنگ‌های هماهنگ  
✅ **تنظیمات شخصی** - سفارشی‌کردن آستانه‌های تحلیل  
✅ **هشدارهای هوشمند** - اطلاعات فوری درباره شرایط بازار

## 📊 شاخص‌های تحلیل

### 1. **روند (Trend)**
- EMA 20, 50, 200
- تشخیص روند صعودی/نزولی/خنثی

### 2. **مومنتوم (Momentum)**
- RSI (14)
- MACD با سیگنال و هیستوگرام

### 3. **حمایت و مقاومت (Support & Resistance)**
- محاسبه خودکار حمایت و مقاومت
- موقعیت قیمت نسبت به این سطح‌ها

### 4. **حجم معاملات (Volume)**
- مقایسه حجم فعلی با میانگین
- تشخیص قدرت روند

## 🎛️ سیستم امتیاز‌دهی

هر تحلیل بر اساس معادله‌ای پیچیده امتیاز می‌شود:

**امتیاز مثبت:**
- +2 روند صعودی
- +2 قیمت نزدیک حمایت
- +1 RSI بین 40-60
- +2 MACD کراس صعودی
- +2 حجم بالا

**امتیاز منفی:**
- -2 قیمت نزدیک مقاومت
- -2 RSI اشباع خرید
- -2 MACD کراس نزولی
- -1 حجم پایین

**نتیجه‌گیری:**
- ✅ امتیاز ≥ 5: **ورود مناسب است**
- ⚠️ امتیاز 3-4: **شرایط متوسط است**
- ❌ امتیاز ≤ 2: **ورود مناسب نیست**

## ⚙️ تنظیمات شخصی

کلیک بر دکمه **⚙️** برای دسترسی به تنظیمات:

- **حد RSI**: تغییر آستانه اشباع خرید (پیش‌فرض: 70)
- **حد امتیاز**: تغییر نیاز برای تصمیم مثبت (پیش‌فرض: 5)
- **حد حجم**: تغییر درصد مورد نیاز برای حجم بالا (پیش‌فرض: 100%)
- **فعال‌سازی شاخص‌ها**: روشن/خاموش کردن هر شاخص
